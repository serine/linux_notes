## How to get a server

- download a debian image
- `cat debian.iso > /dev/sdb`
- `sync`

## First thing first

- `apt-get udpate`
- `apt-get upgrade` 
- `apt-get dist-upgrade`

- select standard installation, no graphical environment (debian)

## Optional install

- ethtool
